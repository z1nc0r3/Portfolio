InternTracker

This web app simplifies the process of managing weekly diaries for internship students. They can easily add, update, and submit their reports to their supervisors. The tool enables students to document their progress and supervisors to provide feedback from a centralized hub.

Features
  • Secure login process for all users.
  • Personalized dashboard for all users.
  • Student can add, update, and submit their weekly diaries.
  • Supervisor can view and provide feedback on student diaries.
  • Evaluators can review the final project diary.
  • Admin can manage students, supervisors and evaluators.
  • Admin can assign supervisors and evaluators for students.
  • Supervisors and Evaluators can generate pdf files of student reports.
  • Bulk user creating feature for Admin.
  • Mobile responsiveness
Tech-Stack
  • Frontend : React JS | MUI | Bootstrap
  • Backend : Laravel | PHP | MySQL
  • Frontend hosted on Vercel.com
  • Backend hosted on Google Cloud VM
